## 一、项目概述

**心镜·AIGC智能体平台** 是一个部署在智慧教室环境中的AI智能体系统。平台以**沐曦MetaX GPU 国产算力**为底座，搭载**moark.com 平台 Qwen3-8B 大模型**作为核心推理引擎，通过**多智能体协作架构**，融合**面部微表情 + VibraImage前庭振动**双模态情绪识别，实现从实时感知、深度心理分析，到**AIGC内容自动生成**的全流程智能化。

### 1.1 一句话理解

> 教室里的"情绪CT机" + "AI心理报告生成器" —— 沐曦国产GPU + moark.com Qwen3-8B大模型 + VibraImage振动分析 + 多Agent协作 + 全流程AIGC。

### 1.2 核心价值

| 维度 | 描述 |
|------|------|
| **AI算力** | 沐曦MetaX GPU国产算力 + moark.com Qwen3-8B大模型，全链路国产化AI |
| **技术创新** | VibraImage前庭振动识别 + 双模态融合 + 多智能体AIGC协作 |
| **社会价值** | 关注青少年心理健康，辅助学校心理工作数字化转型 |
| **开源生态** | 全栈开源（Apache 2.0），推动国产GPU AI生态建设 |
| **产品完整度** | 完整的前后端、Docker部署、API文档、自动化测试 |

## 五、核心模块详解

### 5.1 国产AI算力平台层 (`backend/llm/platform_adapter.py`)

平台为多国产算力平台提供统一适配，支持 moark.com Qwen3-8B 大模型作为主力推理引擎。

**支持的平台**:

| 平台ID | 名称 | GPU/提供商 | 角色 |
|--------|------|-----------|------|
| `lingshu` | moark.com Qwen3-8B | 沐曦MetaX GPU（moark.com） | ★ 主平台 · OpenAI兼容 |
| `gitee_ai` | Gitee.AI | 沐曦 MetaX GPU | 可选 |
| `deepseek` | DeepSeek | DeepSeek | ☆ 备用 |
| `local` | 本地模型 | Ollama / vLLM | 可选 |
| `custom` | 自定义平台 | 任意OpenAI兼容 | 可扩展 |

**关键特性**:
- **moark.com Qwen3-8B**：通过 OpenAI 兼容 API 调用，支持流式推理、reasoning 和 `X-Failover-Enabled` 故障转移。通过 `ExtraBodyChatOpenAI` 子类支持 `top_k`、`frequency_penalty` 等非标准参数
- 平台切换只需修改 `.env` 中的 `AI_PLATFORM=lingshu`
- 自动 Fallback：主平台不可用时自动切换备用平台
- 全部平台国产化（`is_domestic: True`）
- **AIGC 生成器升级**：四个生成器（报告/方案/沟通函/叙事）优先使用 LLM 直接生成，API 不可用时自动降级模板模式，确保系统在任何情况下均可运行

### 5.2 VibraImage前庭振动引擎 (`backend/vibraimage/`)

基于 **Viktor Minkin "Vibraimage, Cybernetics and Emotions" (2020)** 专著公开公式，实现从视频帧到情绪参数的全链路计算。

**处理流水线**:
```
视频 → YOLOv8人脸检测 → 头部ROI裁剪(224×224)
     → 逐像素帧差分 → 过零率频率分析
     → 频率直方图统计 → 左右空间对称性分析 → 频谱功率分布
     → E1-E12情绪参数 → K值心理状态指标
```

**E1-E12 参数体系**:

| 参数 | 名称 | 物理含义 | 计算维度 |
|------|------|---------|---------|
| E1 | 攻击性(Aggression) | 振动频率峰值×离散度 | 统计特征 |
| E2 | 压力(Stress) | 左右半脸运动不对称度 | 空间特征 |
| E3 | 紧张(Tension) | 高频振动能量占比 | 频率特征 |
| E4 | 可疑度(Suspect) | (E1+E2+E3)/3 | 派生 |
| E5 | 平衡(Balance) | 各参数窗口间变异性 | 时间聚合 |
| E6 | 魅力(Charm) | 左右侧运动协调性 | 空间聚合 |
| E7 | 活力(Energy) | 直方图统计综合 | 统计 |
| E8 | 自我调节(Self-Reg) | E5/E6稳定性比率 | 派生 |
| E9 | 抑制(Inhibition) | F1主导周期/总时长 | 时间 |
| E10 | 神经质(Neuroticism) | E9窗口间标准差×10 | 时间 |
| E11 | 抑郁(Depression) | σ相对于0.5Hz偏移 | 病理 |
| E12 | 幸福(Happiness) | 信息效能×能量特征 | 综合 |

**K值**：综合心理状态指标，公式 `K = Σ m_i × (T_i − T_n_i)`，基于10,266人常模数据库。|K|<3正常，3≤|K|<6关注，|K|≥6预警。

**容错设计**：YOLOv8不可用时自动降级Haar Cascade；引擎整体不可用时自动降级随机stub，不影响系统运行。

**E1-E12 → Valence/Arousal 映射**：正性参数（balance/charm/energy/happiness等）→ 正效价，负性参数（aggression/stress/tension/depression等）→ 负效价。Arousal 由 energy + tension 组合计算。

### 5.3 双模态情绪融合 (`backend/tools/emotion_recognition.py`)

```
面部微表情识别 (外部API / stub)
        +
VibraImage前庭振动 (真实物理引擎)
        ↓
加权融合: Valence = 0.6×facial + 0.4×vestibular
         Arousal = 0.6×facial + 0.4×vestibular
        ↓
置信度差异>35% → 自动复核
```

### 5.4 多智能体协作系统 (`backend/agents/`)

5个智能体，每个是独立的 LangGraph ReAct Agent，由 moark.com Qwen3-8B 提供推理能力。

| # | 智能体 | 职责 | 工具 |
|---|--------|------|------|
| 1 | 👁️ 感知智能体 | 双模态情绪识别与融合 | EmotionRecognitionTool |
| 2 | 🧠 分析智能体 | 12项心理指标 + 风险判定 | MentalHealthAnalysisTool |
| 3 | ✨ AIGC报告智能体 | 日报/干预方案/沟通函/叙事 | 4个AIGC生成器 |
| 4 | 🔔 预警智能体 | 三级预警 + 7渠道反馈 | MultiChannelFeedbackTool |
| 5 | 🎯 协调智能体 | 多Agent调度 + SSE推送 | — |

### 5.5 AIGC内容生成引擎 (`backend/aigc/`)

moark.com Qwen3-8B 大模型驱动的 AIGC 生成，在心理健康场景下具备出色的中文理解和生成能力。

**AIGC 生成策略升级（v2.1）**：
- **LLM 优先模式**：通过 `backend/aigc/llm_client.py` 直接调用 moark.com OpenAI 兼容 API，Qwen3-8B 生成高质量的个性化内容
- **模板降级模式**：当 LLM 不可用时（无 API Key / 网络异常 / 调用失败），自动降级到预定义模板填充，确保系统 100% 可用
- 四个生成器均实现了 `_try_llm_generate()` → 模板 fallback 的双轨策略

| 生成器 | 输出 | 应用场景 | 生成策略 |
|--------|------|---------|---------|
| `ReportGenerator` | 心理评估日报/周报 | 每日自动生成 | LLM优先 → 模板降级 |
| `PlanGenerator` | 个性化干预方案 | 黄/红色预警 | LLM优先 → 模板降级 |
| `LetterGenerator` | 家校沟通函（三级措辞） | 家长通知 | LLM优先 → 模板降级 |
| `NarrativeGenerator` | 学生成长叙事 | 学期评估 | LLM优先 → 模板降级 |

## 九、关键设计决策与创新点

| 决策 | 方案 | 理由 |
|------|------|------|
| 主推理模型 | **moark.com Qwen3-8B** | OpenAI 兼容 API，支持流式推理 + reasoning，中文生成能力强 |
| 前庭振动识别 | **VibraImage白盒引擎** | 基于公开公式，E1-E12物理含义明确，非黑盒神经网络 |
| 双模态融合 | **面部微表情(0.6) + 前庭振动(0.4)** | 互补校验，置信度差异>35%触发复核 |
| 多智能体协作 | **5 Agent + ReAct模式** | 职责分离，独立优化，Qwen3-8B 驱动推理 |
| 平台适配 | **ExtraBodyChatOpenAI** | 支持 moark.com 非标准参数（top_k, frequency_penalty, X-Failover-Enabled），架构可扩展 |
| AIGC实现 | **LLM 优先 + 模板降级** | Qwen3-8B 直接生成高质量内容，不可用时自动回退模板 |

**核心创新点**:

1. **沐曦国产GPU + moark.com Qwen3-8B 全链路**：从底层算力到上层推理引擎全部国产化，通过 OpenAI 兼容 API 调用，支持流式推理、reasoning 和故障转移
2. **VibraImage白盒前庭振动分析**：基于1980-2020年专著公式体系，12维情绪参数每个都有明确物理含义
3. **LLM 驱动的 AIGC 双轨生成**：Qwen3-8B 直接生成心理报告/干预方案/沟通函/成长叙事等个性化内容，不可用时自动降级模板模式，兼顾质量与可靠性
4. **真正的多智能体协作**：5个Agent分工明确，Qwen3-8B 驱动ReAct推理循环，SSE流式展示全过程
5. **产品级完整度**：前后端、Docker、API文档、自动化测试全覆盖
